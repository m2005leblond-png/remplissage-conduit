    📱 Guide d'installation hors ligne

    Ajoutez cette application sur l'écran d'accueil de votre appareil pour pouvoir l'utiliser directement sur les chantiers sans aucune connexion réseau.

    🍏 Apple iOS (iPhone / iPad)

    1. Ouvrez ce site dans le navigateur Safari.
    2. Appuyez sur le bouton de partage en bas de l'écran (l'icône avec un carré et une flèche vers le haut).
    3. Faites défiler les options et sélectionnez Sur l'écran d'accueil.
    4. Appuyez sur Ajouter en haut à droite.

    🤖 Android (Chrome)

    1. Ouvrez ce site dans l'application Google Chrome.
    2. Appuyez sur les 3 petits points de menu en haut à droite.
    3. Sélectionnez l'option Installer l'application ou Ajouter à l'écran d'accueil.

    💻 Ordinateur (Chrome / Edge)

    1. Cliquez sur l'icône d'installation (un petit écran avec une flèche) située tout à droite dans la barre d'adresse.
    2. Confirmez en cliquant sur le bouton Installer.
