    📱 Guide d'installation PWA

    Ajoutez cette application sur votre écran d'accueil pour pouvoir l'utiliser sans réseau directement sur les chantiers.

    🍏 Apple iOS (iPhone / iPad)

    1. Ouvrez ce site dans le navigateur Safari.
    2. Appuyez sur le bouton de partage en bas de l'écran ⎋.
    3. Faites défiler le menu et sélectionnez Sur l'écran d'accueil ⊞.
    4. Appuyez sur Ajouter en haut à droite.

    🤖 Android (Chrome)

    1. Ouvrez ce site dans Google Chrome.
    2. Appuyez sur les 3 petits points ⋮ en haut à droite.
    3. Sélectionnez l'option Installer l'application ou Ajouter à l'écran d'accueil.

    💻 Ordinateur (Chrome / Edge)

    1. Dans la barre d'adresse en haut, cliquez sur l'icône d'installation ⨁ située à droite.
    2. Cliquez sur le bouton Installer.
